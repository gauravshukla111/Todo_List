# 📝 Todo App

A simple and modern Todo List application built using HTML, CSS, and JavaScript.

## 🚀 Live Demo
👉 https://todo-list-six-lyart-58.vercel.app/

## ✨ Features
- Add new tasks
- Edit tasks ✏️
- Delete tasks ❌
- Mark tasks as completed ✅
- Clear all tasks
- Data saved using LocalStorage
- Smooth animations and modern UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📌 About Project
This project is built as part of my frontend revision to strengthen core concepts like DOM manipulation, events, and localStorage.
